# CSS Clock

A simple static clock design built using **HTML and CSS** only.

![Clock Screenshot](clock-screenshot.png)
---
## Features
- Pure HTML & CSS (no JavaScript)
- Clock face with decorative hands and frame
- Simple and clean design
---
## Files
- `index.html` — structure of the clock
- `style.css` — all visual styling
---
## How to view
Open `index.html` in your browser to see the clock design.
